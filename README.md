## Social_Media_App

> Stack Used:  MERN
>
> Techonolgy Used:
>
> * Javascript/Node.js
> * Mongo DB
>
> Libraries Used:
>
> 1. React Js
> 2. Express Js
> 3. Bootstrap
> 4. Mongoose
> 5. Multer
> 6. Nodemailer
>
> Scope:
>
> * Authentication:
>   * JWT authentication.
>   * Password ecnryption using bcryptjs.
>   * Reset password using reset password link sent on registered mail.
> * User:
>   * Can Register/Login.
>   * Can reset password using password reset mail.
>   * Can Post, Like & Comment.
